# SnakeGame
Snek gem
<h4>Linux;</h4>

1. ```git clone https://github.com/helloimalemur/SnakeGame```

2. ```cd SnakeGame/```

3. ```./gradlew run```

<h4>Windows;</h4>

1. <a href="https://github.com/helloimalemur/SnakeGame/archive/refs/heads/master.zip">Download ZIP</a>

2. Extract zip

3. run in PowerShell: ```.\gradlew.bat run```
